The source code for [debgen.jcmhost.com.br/](https://debgen.jcmhost.com.br/)

# Setup

## Install source code

```
git clone https://github.com/jeanmajolo/debgen.git 
```

## Install packages

```
yarn
```

or

```
npm install
```

## Starting dev server

### Dev mode

```
yarn dev
```

## Building code

```
yarn build
```
